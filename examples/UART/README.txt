This example demonstrates UART1 functionality. UART is initialized at 9600 baud.
Baudrate is defined in stm8/uart.h (stable up to 9600 with 2MHz internal oscillator).

 --------------------
|                    |
|                    |
|                    |
|                    |
|                    |
|       <STM8>       |
|                    |
|                PD5 |-> TX
|                    |
|                PD6 |-> RX
|                    |
 --------------------
